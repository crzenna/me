Business systems analysis and design

Year 2 module

Module description

The module content includes the following:

  • Introduction to systems theory
  • Model building strategies
  • Analysis and design
  • Introduction to soft systems analysis
  • Introduction to systems dynamics methodology
  • Systems modelling using system dynamics
  • Soft systems approach to systems design and implementation
  • Human factors in the design and implementation of information systems in an organizational context
