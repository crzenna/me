Programming for media and design

Year 1 module

Module description

This is an introduction to computer programming for interactive media applications. Students will design, write, test and debug programs. They will learn the essentials of computer programming: variables, data types and data structures and programming constructs such conditionals, loops and functions. They will also be introduced to the basics of object-oriented programming.These programming fundamentals will be used to develop interactive applications to process digital images and to implement animations.
